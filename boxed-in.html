Boxed In:
A C Project

Play the game here. Press the 'Run' button.
I completed this project in my first year of university (2023), applying many fundamental coding principals, including:

  • Defensive Programming - protects against invalid access or logic errors
  • Structured Programming - procedural, clear control flow
  • Mutable State + Encoded History - tracks and manipulates game state + supports undoModularity
  • Modularity - clear function separation for each task
  • Abstraction - reusable helper functions to simplify code
  • Readability - constants & naming to improve understanding of game state


Note: This was a graded university project - some code was provided by the University of Auckland.
All original code authored by me is clearly commented.
